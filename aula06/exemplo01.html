Seletores

Seletores simples

Seletores simples são aqueles que selecionam elementos com base em um único critério, como o nome da tag, a classe ou o ID.

Por tipo

Seleciona todos os elementos de um determinado tipo.

Exemplo de parágrafo não adjacente a um elemento h3.

Testando parágrafo!

Por classe

Seleciona todos os elementos que possuem uma determinada classe.

Por ID

Seleciona um único elemento que possui um determinado ID.

Seletores compostos

Seletores compostos são aqueles que combinam múltiplos seletores simples para selecionar elementos com base em mais de um critério.
